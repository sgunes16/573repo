### 1.2 Core Concept

- **Give an hour** â†’ Earn 1 time credit

| Feature | Description |

## 5. Handshake (Exchange) System

The **Handshake** is the core feature of The Hiveâ€”it's how two users connect and complete a service exchange.

### 5.1 Starting a Handshake

**As a Requester (for an Offer):**
1. Browse and find an offer you want
2. Click **"Start Handshake"** on the offer detail page
3. A new handshake is created in **PENDING** status
4. For 1-to-1 offers: Propose a date/time
5. Wait for the provider to accept

**As a Requester (for a Want):**
1. Find a want you can fulfill
2. Click **"Start Handshake"**
3. You become the **Provider** (even though you initiated)

### 5.2 Handshake Progress Steps

**For 1-to-1 Exchanges:**
```
[Requested] â†’ [Date Set] â†’ [Accepted] â†’ [Done]
```

**For Group Exchanges:**
```
[Requested] â†’ [Accepted] â†’ [Done]
```

### 5.3 Handshake Statuses

| Status | Description |
|--------|-------------|
| **PENDING** | Request sent, awaiting response |
| **ACCEPTED** | Provider accepted, exchange in progress |
| **COMPLETED** | Both parties confirmed completion |
| **CANCELLED** | Requester cancelled the request |
| **REJECTED** | Provider declined the request |

### 5.4 Handshake Actions

**As Provider (receiving a request):**
- **Accept**: Agree to the exchange
- **Reject**: Decline the request

**As Requester:**
- **Propose Date**: Set when you want the exchange (1-to-1 only)
- **Cancel Request**: Withdraw your request

**After Acceptance:**
- **Mark Complete**: Confirm the exchange happened
- Both parties must confirm for completion

### 5.5 Chat Feature

Every handshake includes a real-time chat:
- Coordinate details with the other party
- Discuss location, timing, or specifics
- Messages are saved in the exchange

### 5.6 Rating System

After completing an exchange, rate the other party:

| Criteria | Range |
|----------|-------|
| **Communication** | 1-5 stars |
| **Punctuality** | 1-5 stars |
| **Would Recommend** | Yes/No |
| **Comment** | Optional text |

## 10. Reporting & Safety

### 10.1 When to Report

Report content or users that:
- Contain spam
- Are inappropriate
- Appear to be fake profiles
- Involve harassment
- Seem fraudulent

### 10.2 How to Report

**Report an Offer/Want:**
1. Open the offer/want detail page
2. Click the **"Report"** button
3. Select a reason
4. Add description (optional)
5. Submit

**Report a User:**
1. Visit their profile
2. Click **"Report User"**
3. Select reason and submit

**Report an Exchange:**
1. Open the handshake page
2. Click **"Report Exchange"**
3. Provide details

### 10.3 Report Reasons

| Reason | Description |
|--------|-------------|
| **Spam** | Unsolicited advertising |
| **Inappropriate** | Offensive content |
| **Fake Profile** | Impersonation or fake identity |
| **Harassment** | Abusive behavior |
| **Fraud** | Deceptive practices |
| **Other** | Other concerns |

### 10.4 What Happens After Reporting

1. Report is queued for admin review
2. Admin investigates the report
3. Actions may include:
   - Content removal
   - User warning
   - Account suspension
4. You'll be notified of significant actions

## 12. Troubleshooting & FAQ

### 12.1 Frequently Asked Questions

**Q: How do I earn more time credits?**
> A: Provide services to other community members. Each hour of service earns 1 time credit.

**Q: What if I don't have enough credits to request a service?**
> A: You need sufficient available credits before requesting. Earn credits by offering your skills first.

**Q: Can I cancel an exchange after it's accepted?**
> A: Yes, requesters can cancel until the provider marks it complete. Credits are returned.

**Q: How is my rating calculated?**
> A: Your rating is the average of all ratings received from completed exchanges.

**Q: Why can't I create posts in the forum?**
> A: You must verify your email and not be banned to create forum posts.

**Q: What happens to my credits if an exchange is cancelled?**
> A: Blocked credits are returned to your available balance.

**Q: Can I edit an offer after someone requested it?**
> A: Yes, you can edit your offers. Active exchanges continue with original terms.

**Q: How do group offers work?**
> A: Group offers have multiple slots. Each participant joins individually and confirms completion separately.

**Issue: Location not showing on map**

**Issue: Not receiving email verification**

**Issue: WebSocket connection failed**

> 3. Try a smaller image
